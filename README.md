# Trabajo Final - Gestión de Inventario (Grupo A5)

Este proyecto es una aplicacióndiseñada para gestionar el inventario de un laboratorio. Permite administrar tanto equipos técnicos como materiales consumibles, controlando existencias y alertas de seguridad.

##  Propósito del Proyecto
El objetivo es centralizar el control de los activos del laboratorio. El sistema permite:
* **Clasificar objetos:** Diferenciar entre equipos (que requieren estado operativo) y consumibles (que requieren control de lote).
* **Control de Caducidad:** El sistema detecta automáticamente si un lote ha vencido comparando la fecha actual con la del objeto.
* **Automatización de datos:** Evitar errores manuales al registrar parámetros específicos como RPM, rangos térmicos o errores de medida.
